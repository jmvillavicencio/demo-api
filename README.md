# demo-api

> Simple APi demo

## Dependencies
* MongoDB running on port 27017

## Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:4040
$ yarn run dev

```
## Config Setup
So far only the configuration for development is set.
To modify it, change the file `.env.development`
